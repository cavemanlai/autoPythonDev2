作业1：登录接口
    输入用户名密码
    验证成功后显示欢迎信息
    输错三次后锁定
---------------------------------------------------------
脚本说明：
    unlock.py   用来初始化账号密码以及解锁账号
    login_interface.py  用户登录后，可以选择玩猜数字游戏

    三级菜单
    可依次选择进入各子菜单
    所需新知识点：列表、字典
-------------------------------------------------------------
脚本说明：
    muti_menu.py
    运行显示城市信息，选择对应的索引号进入该城市的区信息列表，
    在区信息列表中，选择对应的区号索引，即可查询该区的地名。
